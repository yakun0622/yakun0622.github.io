calibre-web 豆瓣API失效解决

网上常见的 apikey 0df993c66c0c636e29ecbb5344252a4a 已经在八月被豆瓣封禁。
所以需要将旧的key改为054022eaeae0b00e0fc068c0c0a2102a

修改方式：

vim /calibre-web/app/cps/static/js/get_meta.js

找到

// 找到：
var apikey = "0df993c66c0c636e29ecbb5344252a4a";
// 改为：
var apikey = "054022eaeae0b00e0fc068c0c0a2102a";
